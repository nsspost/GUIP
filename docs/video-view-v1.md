# Video View V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Экран нужен для попытки показать видеосигнал локально или отразить, что идет вывод через USB.

## Принцип

Этот экран должен честно показывать одно из трех состояний:
- видео отображается;
- идет попытка захвата/декодирования;
- видео недоступно.

## Базовый Вид При Рабочем Видео

```text
+--------------------------------+
| 12:46 BAT76 VOL2 VIB SIMPLE   |
+--------------------------------+
| VIDEO VIEW                    |
| Analog video / 1.2 GHz        |
+--------------------------------+
|                                |
|        [ ВИДЕОКАДР ]           |
|                                |
|                                |
|                                |
|                                |
+--------------------------------+
| Signal: -51 dBm   USB: ON      |
| Stream: active                 |
+--------------------------------+
| OK: действия                   |
| Назад: сигнал   Fn: mute       |
+--------------------------------+
```

## Вид При Отсутствии Видео

```text
+--------------------------------+
| 12:46 BAT76 VOL2 VIB SIMPLE   |
+--------------------------------+
| VIDEO VIEW                    |
| Analog video / 1.2 GHz        |
+--------------------------------+
|                                |
|     Видео недоступно           |
|                                |
|  Сигнал есть, но картинки нет  |
|                                |
|                                |
+--------------------------------+
| Signal: -63 dBm                |
| Проверьте источник/режим       |
+--------------------------------+
| OK: назад                      |
| Назад: сигнал                  |
+--------------------------------+
```

## Действия

- `Назад к сигналу`
- `Вернуться к сканированию`
- `Повторить попытку`

## Данные

- `signal_name`
- `frequency_text`
- `power_dbm`
- `video_state`
- `usb_state`
- `last_frame_available`
