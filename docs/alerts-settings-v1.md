# Alerts Settings V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Экран управляет тем, как прибор уведомляет пользователя.

## Базовый Вид

```text
+--------------------------------+
| ОПОВЕЩЕНИЯ                    |
+--------------------------------+
| > Звук                ON       |
|   Вибрация            ON       |
|   Громкость           2/5      |
|   Тревога для новых   ON       |
|   Тревога для слабых  OFF      |
+--------------------------------+
| OK: изменить                   |
| Назад: меню                    |
+--------------------------------+
```

## Принцип

Все должно настраиваться быстро и без длинных подменю.
