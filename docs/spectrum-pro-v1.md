# Spectrum Pro V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Это основной спектральный экран для продвинутого режима.
Он нужен для более детального исследования эфира.

## Основные Возможности

- просмотр спектра;
- zoom;
- сдвиг влево/вправо;
- выделение участка интереса;
- быстрый переход к удержанию на частоте.

- `zoom`
- `pan`
- `hold on selected region`
- `return`
