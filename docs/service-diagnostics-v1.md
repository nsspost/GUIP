# Service Diagnostics V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Экран для сервисной и технической информации.
Не должен быть частью обычного сценария `Simple`.

## Что Показывать

- версия прошивки;
- состояние USB;
- состояние видео;
- число обнаружений;
- базовые технические флаги и счетчики.

## Базовый Вид

```text
+--------------------------------+
| СЕРВИС / ДИАГНОСТИКА          |
+--------------------------------+
| FW: 0.1.0                     |
| USB: active                   |
| Video: idle                   |
| Detections: 124               |
| UVC: ready                    |
| Battery ADC: OK               |
+--------------------------------+
| OK: подробнее                  |
| Назад: меню                    |
+--------------------------------+
```
