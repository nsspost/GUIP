# Hold Monitor V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Этот экран открывается, когда пользователь выбирает действие `Остаться на частоте`.
Он нужен для наблюдения за одним конкретным сигналом во времени.

## Что Пользователь Должен Понять

- прибор больше не в общем поиске, а удерживается на выбранной частоте;
- уровень сигнала сейчас растет, падает или стабилен;
- можно быстро вернуться к общему сканированию.

## Базовый Вид

```text
+--------------------------------+
| 12:45 BAT77 VOL2 VIB SIMPLE   |
+--------------------------------+
| УДЕРЖАНИЕ ЧАСТОТЫ             |
| Analog video                  |
+--------------------------------+
| Частота: 1.2 GHz              |
| Текущий уровень: -49 dBm      |
| Динамика: растет              |
+--------------------------------+
|        УРОВЕНЬ ВО ВРЕМЕНИ      |
|      __---__                  |
|   ---      ---__              |
| _--            --_            |
|                                |
+--------------------------------+
| Статус: сигнал активен         |
| Время удержания: 00:02:14      |
+--------------------------------+
| OK: действия                   |
| Назад: сканирование            |
+--------------------------------+
```

## Действия

Контекстное меню отсюда:
- `Вернуться к сканированию`
- `Показать видео / USB`
- `Игнорировать сигнал`

## Данные

- `type_name`
- `frequency_text`
- `power_dbm`
- `trend`
- `hold_duration`
- `small_history_series`
