# Ignored Signals V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Этот экран показывает сигналы или правила, для которых уведомления отключены.

## Базовый Вид

```text
+--------------------------------+
| ИГНОРИРУЕМЫЕ СИГНАЛЫ          |
+--------------------------------+
| > Analog video / 1.2 GHz      |
|   Beacon / 433 MHz            |
|   Unknown / weak stable       |
+--------------------------------+
| OK: открыть                    |
| Назад: меню   Fn: удалить      |
+--------------------------------+
```

## Действия

Для записи:
- `вернуть в наблюдение`
- `удалить правило`
- `назад`

## Принцип

Здесь важно не прятать отключенные сигналы слишком глубоко, чтобы пользователь мог быстро исправить ошибочное игнорирование.
