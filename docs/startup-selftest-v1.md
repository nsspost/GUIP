# Startup Selftest V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Экран нужен для старта прибора, короткой индикации загрузки и самотеста.

## Основная Логика

- не держать пользователя здесь слишком долго;
- показывать понятный прогресс;
- в случае ошибки показывать, что именно не прошло.

## Базовый Вид Загрузки

```text
+--------------------------------+
|                                |
|            GUIP                |
|                                |
|      Инициализация прибора     |
|                                |
|         [#######....]          |
|                                |
|      Проверка модулей...       |
|                                |
|                                |
+--------------------------------+
```

## Вид При Ошибке Самотеста

```text
+--------------------------------+
| САМОТЕСТ                      |
+--------------------------------+
| Ошибка инициализации видео    |
| Остальные модули: OK          |
|                                |
| Прибор может продолжить       |
| без видеорежима               |
+--------------------------------+
| OK: продолжить                |
| Назад: сервис                 |
+--------------------------------+
```
