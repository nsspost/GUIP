# Signal Types Settings V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Экран позволяет включать и выключать интересующие типы сигналов.

## Базовый Вид

```text
+--------------------------------+
| ТИПЫ СИГНАЛОВ                 |
+--------------------------------+
| > Analog video         ON      |
|   Digital burst        ON      |
|   Beacon               ON      |
|   Unknown              OFF     |
|   Experimental         OFF     |
+--------------------------------+
| OK: переключить                |
| Назад: меню                    |
+--------------------------------+
```

## Логика

- короткие названия;
- включение и выключение по месту;
- длинные объяснения не нужны.
