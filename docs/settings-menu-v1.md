# Settings Menu V1

Обновлено: 2026-04-30
Статус: черновая экранная спецификация

## Назначение

Главный экран меню настроек должен быть коротким, иерархичным и без перегруза.

## Основные Разделы

- Режим интерфейса
- Типы сигналов
- Оповещения
- Экран
- Игнорируемые сигналы
- Видео / USB
- Сервис

## Базовый Вид

```text
+--------------------------------+
| 12:50 BAT75 VOL2 VIB SIMPLE   |
+--------------------------------+
| МЕНЮ                          |
+--------------------------------+
| > Режим интерфейса            |
|   Типы сигналов               |
|   Оповещения                  |
|   Экран                       |
|   Игнорируемые сигналы        |
|   Видео / USB                 |
|   Сервис                      |
+--------------------------------+
| OK: открыть                   |
| Назад: выйти                  |
+--------------------------------+
```

## Принцип

В `Simple` режиме здесь должны жить только понятные человеку разделы.
Глубокие инженерные настройки лучше уводить в `Pro` или `Сервис`.
